# Lumen - macOS 개인용 런처 앱 개발 계획

## 프로젝트 개요
macOS에서 Raycast를 대체할 **개인용** 런처 앱. 배포/상품화 계획 없음. 혼자 쓸 것이므로 국제화, 설정 UI, 메뉴바 아이콘, 애니메이션 등 불필요. MVP로 빠른 기능 개발이 최우선.

## 기술 스택
- Swift + SwiftUI
- macOS 전용 (최소 버전: 현재 사용 중인 macOS)
- Xcode 프로젝트

## 핵심 원칙
- **빠른 기능 개발 최우선**: 애니메이션, 시각 효과 등 꾸미기 일체 금지
- **미니멀**: 메모리 적게, 가볍게
- **다크모드만** 지원
- **설정 UI 없음**: 변경 필요 시 코드 수정 후 재실행
- **하드코딩 OK**: API 키, 단축키 등 코드에 상수로
- **과도한 추상화 금지**: 다만 내장 기능은 프로토콜 기반으로 구조화 (추가/제거 용이하게)

## UX 흐름

### 메인 검색창
- `Option + Space`로 호출 (코드에서 변경 가능)
- 화면 중앙 상단에 Raycast 크기의 플로팅 검색창
- ESC 또는 포커스 잃으면 닫힘

### 검색 동작
검색창에 타이핑하면 **앱만 검색**됨.
결과 목록에는 설치된 앱과 내장 기능이 함께 표시:

```
┌──────────────────────────────────┐
│ 🔍 검색어 입력...                  │
├──────────────────────────────────┤
│  Chrome                    앱   │
│  Calculator                앱   │
│  VS Code                   앱   │
│  ─────────────────────────────  │
│  🌐 AI 번역               내장  │
└──────────────────────────────────┘
```

- **설치된 앱**: Enter로 바로 실행
- **내장 기능**: Enter로 선택 시 별도 윈도우가 열리고, 메인 검색창은 닫힘
- 한 화면에 5개 표시, 키보드 ↑↓로 스크롤
- 내장 기능도 검색 필터링 대상 (예: "번역" 입력 → AI 번역 표시)

### 내장 기능 구조
프로토콜 기반으로 추상화. 하드코딩이지만 새 기능 추가 시 프로토콜 채택만 하면 됨. 주석 처리로 비활성화 가능.

현재 내장 기능 2개:
1. **AI 번역**
2. **윈도우 마그넷** (on/off 가능)

## 기능 상세

### 1. 앱 검색 및 실행
- `/Applications`, `~/Applications`에서 앱 목록 로드
- 실시간 필터링
- Enter로 실행
- 최근/자주 사용 앱 우선 표시

### 2. AI 번역 (내장 기능 - 별도 플로팅 윈도우)
- 내장 기능 선택 또는 `Cmd+Shift+T`로 다이렉트 호출 가능
- 별도 플로팅 윈도우 (Always on top)
- 타이틀 바 자체가 입력란 (컴팩트)
- 열자마자 커서 자동 포커스 → 바로 타이핑 가능
- 한국어 ↔ 영어 (자동 감지)
- 결과 클립보드 복사 가능
- 히스토리 없음
- **OpenAI API** 사용 (API 키 하드코딩)
- 다른 LLM으로 변경 시 그때 추상화

```
┌─────────────────────────────────────┐
│ 🔍 번역할 텍스트 입력...         ✕   │
├─────────────────────────────────────┤
│                                     │
│ 번역 결과가 여기에 표시됩니다          │
│                                     │
└─────────────────────────────────────┘
```

### 3. 윈도우 마그넷 (내장 기능 - 글로벌 단축키)
- on/off 플래그로 제어 (코드에서)
- `Ctrl+Option+←` : 왼쪽 스내핑
- `Ctrl+Option+→` : 오른쪽 스내핑
- 좌우만 지원 (상하/최대화 없음)
- 순환 단계: 20% → 40% → 60% → 80% → 100% → 다시 20%
- 현재 윈도우 크기가 해당 단계에 없으면 20%부터 시작
- AXUIElement API로 활성 윈도우 제어
- 접근성 권한 필요 (글로벌 단축키와 동일 권한)

### 4. 커스텀 단축키
- 내장 기능별 글로벌 단축키 지정
- `Option+Space`: 메인 검색창
- `Cmd+Shift+T`: AI 번역 바로 열기
- `Ctrl+Option+←/→`: 윈도우 마그넷
- 코드에 하드코딩 (변경 시 코드 수정)

### 5. 클립보드 관리 (나중에)
- 현재 Maccy 사용 중이라 후순위
- 나중에 내장 기능으로 추가 가능
